## 视差滚动

对页面内容进行分层，页面滚动时每一个层级滚动速度不同，造成的视觉差会让页面更加立体

通过 css 的 3d 效果实现（容器设置景深，开启 3 的效果，设置 overflow：auto）

```css
.wrapper {
  transform-style: preserve-3d;
  overflow: auto;
  perspective: 1px;
}
.iphone {
  transform: translate3d(-50%, 0, -1px);
}
```
