# JS 防抖节流函数

## 防抖函数

在延迟范围内重复调用只执行最后一次。

- 一参：待执行函数。
- 二参：延迟多久后执行。

```js
function antiShake(fn, interval = 100) {
  let timmer;
  return function(...args) {
    clearTimeout(timmer);
    timmer = setTimeout(fn, interval, ...args);
  };
}
```

- 示例

```js
const consoleArg = antiShake(function(arg) {
  console.log(arg);
});

consoleArg(123); // 123
```
