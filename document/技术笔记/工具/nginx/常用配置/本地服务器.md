# 本地服务器

启动一个本地静态资源服务器，将所有请求映射到 D:\code\h5\hm-admin-web\dist\haoma 目录。

访问 localhost:8888 将展示 D:\code\h5\hm-admin-web\dist\haoma\index.html 页面。

```conf
http {

    server {
        listen       8888;
        root        D:\code\h5\hm-admin-web\dist\haoma;
        server_name  localhost;
        location / {}
    }
}

```
