    # 如何预读一本书

    对于一本陌生的书，快速的浏览一遍可以大致了解作者的主张，并且对书籍的内容有一些基本的认知。预读之后就可以判断这本书是不是符合预期，再决定是否进行深入的阅读。

    不是所有书都有阅读的必要，预读可以帮助节省时间、过滤垃圾信息。

    # 有系统的略读

    有系统的略读，就是准备要了解本书的架构。

    1. 看扉页、序或前言。
    2. 研究目录，大致了解本书的结构和作者想表达的思想。
    3. 从目录中挑选主要章节阅读，重点关注章节摘要和章节结尾内容。
    4. 不要忽略最后两三页，通常作者会在书籍的结尾将自认为重要的观点重新整理一遍。

    # 粗浅的阅读

    粗浅的阅读是阅读者想要了解全书内容的第一个必要步骤。

    对于难以理解的书可以在第一遍阅读时忽略掉不能理解的内容，专注于可以理解的内容。这些内容会在重读第一次略过的那部分时，增进理解。就算不重读，对一本高难度的书了解了一半，也比什么都不了解要好些。

    如果在一碰上困难的地方就停住，最后就可能对这本书真的一无所知了。

    第一遍阅读时遇到的不理解的部分可以先记录下来，阅读一遍后再终点关注。

    # 如何主动阅读（分析阅读）

    任何一种超越基础阅读的阅读层次，核心都是要不断地提出问题，并且尝试去回答这些问题。

    读一本书时，一定要提出以下四个问题：

    1. 整体来说，这本书到底在谈些什么，在它所涉及到的内容中没有谈到的内容是哪些？
    2. 作者的观点是什么，他是如何阐述自己的观点的？
    3. 这本书说的有道理吗，有没有不合理的地方？
    4. 这本书跟你有什么关系？
      1. 这本书的内容对你有意义吗？它是否对你产生了启发。
      2. 如果一本书给告诉了你一些知识，问问自己这些知识真的是自己需要的吗？

    # 找出一本书在谈些什么

    1. 依照书本的种类与主题分类。
    2. 用最简短的句子说出整本书在说什么。
    3. 按照顺序与关系，列出全书的重要部分。将全书的纲要拟出来之后，再讲各个部分的纲要也一一列出。
    4. 找出作者在问的问题，或作者想要解决的问题。

    # 提出自己的见解

    一个人如果不能发动自己的独立判断力对事务做出自己的判断，他根本就学不到任何东西。最能学习的读者，也就是最能批评的读者。

    确定作者是否解决了自己提出的问题，解答是否正确。

    # 如何记笔记

    一个人如果说他知道他在想什么，却说不出来，通常是他其实并不知道自己在想些什么。

    1. 列出书本内容的大纲和重要信息条目。
    2. 阐述全书的大意。

    # 阅读技巧

    • 把手指作为指针，顺着文字一行一行的移动下去，一旦你的眼睛能跟着手移动时，就能读到这些句子了。这样可以提高注意力，避免眼神乱飘。
