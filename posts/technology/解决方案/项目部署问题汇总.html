    # ng build 时出现 FATAL ERROR: Ineffective mark-compacts near heap limit Allocation failed - JavaScript heap out of memory

    问题原因：js 运行时内存不足。

    解决方案：在 ng build 之前执行 NODE_OPTIONS=--max-old-space-size=4096 提高内存。
