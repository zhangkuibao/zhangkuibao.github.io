    # postMessage【已完成】（2023-07-21）

    postMessage 是实现跨站通信的一种手段，通信对象只能是从当前页面打开的子页面（iframe、window.open）。

    • 发送消息

      otherWindow.postMessage(message, targetOrigin, [transfer]);

      otherWindow 是一个其他窗口的引用，比如 iframe 的 contentWindow 属性、执行 window.open 返回的窗口对象、或者是命名过或数值索引的 window.frames。

    • 接收消息

      window.addEventListener("message", (event) => {
  var origin = event.origin || event.originalEvent.origin; // 发送消息的源网站
});
